# spring-boot-postgres-vue-docker

## Stack
- Spring Boot 3
- Spring Data 3
- Postgres
- Docker
- Maven
- Tomcat

## Install
To build services and launch them in containers, you need to run `start.sh`.
```bash
$ bash start.sh
```
